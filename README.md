# BetterWMS
Leaflet WMS + GetFeatureInfo

Forked from https://github.com/jfaraklit/BetterWMS

## Usage

```script
L.tileLayer.betterWms(wms url, {
        layers: "layer",
        format: 'image/png',
        transparent: "true",
        style: 'style',
        opacity: 1,
        attribution: "attribution",
    });
